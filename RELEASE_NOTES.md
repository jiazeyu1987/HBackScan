## 🚀 快速开始

### 环境要求
- Python 3.12+
- Docker & Docker Compose
- 4GB+ RAM
- 2GB+ 磁盘空间

### 安装步骤
```bash
# 1. 克隆项目
git clone <repository-url>
cd hospital-scanner

# 2. 使用Docker运行
docker-compose up -d

# 3. 或者本地运行
pip install -r requirements.txt
python main.py
```

### 验证安装
```bash
# 运行测试
pytest tests/

# 运行验收测试
python run_acceptance_tests.py
```
