# GET `/`

- 入口：`code/hospital_scanner/main.py:99`
- 作用：返回硬编码的服务描述（message/version/status）。

## 执行流程
1. 路由命中后直接返回常量字典，不依赖任何组件。

## 设计原因
- 作为最小化的存活探针/欢迎页，快速确认应用已启动。

## 优点
- 无外部依赖，响应极快，几乎不会失败。

## 缺点与风险
- 版本/状态为硬编码，无法反映真实部署或依赖健康。
- 缺少诊断信息（启动时间、依赖状态），排障价值低。

## 改进建议
- 版本从单一配置源读取（如 git tag 或 package 元数据）。
- 可附带构建哈希、启动时间或健康检查链接，提高可观测性。
